Веб-додаток інтелектуального бронювання місць


Опис проєкту  
Проєкт реалізує веб-додаток для перегляду подій та бронювання місць з використанням інтелектуальних рекомендацій.
Система побудована за клієнт-серверною архітектурою та складається з серверної частини на базі Django, клієнтського вебінтерфейсу та модуля рекомендацій.  
Перегляд подій доступний без реєстрації.
Бронювання місць та персоналізовані рекомендації доступні для зареєстрованих користувачів.


Використані технології  

Серверна частина  
-Python  
-Django  
-Django REST Framework  
-SQLite  
-Pandas  
-Scikit-learn  

Клієнтська частина  
-Next.js
-React  
-JavaScript / TypeScript  
-Tailwind CSS  

Структура проєкту  
bookings – робота з бронюванням місць  
users – користувачі та реєстрація  
shows – події та інформація про них  
ml – логіка інтелектуальних рекомендацій  
frontend – сторінки сайту та інтерфейс  
manage.py – запуск серверної частини  


Інструкція зі збірки та запуску
1. Встановити Python, Node.js та Git.
2. Клонувати репозиторій з проєктом:
git clone https://github.com/VixToryH/Theatre-booking-system
3. Перейти у кореневу директорію проєкту.

Запуск серверної частини

python -m venv venv  
venv\Scripts\activate   
pip install -r requirements.txt  
python manage.py migrate  
python manage.py runserver  

Сервер буде доступний за адресою
http://localhost:8000

Запуск клієнтської частини

cd frontend  
npm install  
npm run dev  

Клієнтський інтерфейс буде доступний за адресою  
http://localhost:3000

Особливості реалізації  

Реалізовано інтелектуальне бронювання місць з використанням рекомендацій.  
Система має клієнт-серверну архітектуру.  
Серверна та клієнтська частини запускаються окремо.  
Рекомендації формуються на основі історії бронювань та жанрів подій.
